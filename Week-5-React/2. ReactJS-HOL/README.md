This repository contains the multi-component implementation of the "studentapp" project.
The project utilizes modular React Class Components (`Home`, `About`, and `Contact`) inside a dedicated directory.
Each component dynamically renders its respective portal welcome message onto the single-page interface.
Run `npm install` to load configuration dependencies, followed by `npm start` to run locally.
The active application runs on the local server interface at `http://localhost:3000`.
