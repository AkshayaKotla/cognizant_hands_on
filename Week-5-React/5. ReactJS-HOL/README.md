Academy Cohort Dashboard (CohortDashboard)
This repository contains the CSS Module implementation of the "cohortdashboard" project.
It demonstrates locally scoped layout components using `CohortDetails.module.css`.
The dashboard renders cohort boxes dynamically using CSS classes imported via JavaScript object styles.
Status header elements apply real-time condition-based colors: green for ongoing and blue for completed status.
Run `npm install` and `npm start` to open the styled portal interface locally.
