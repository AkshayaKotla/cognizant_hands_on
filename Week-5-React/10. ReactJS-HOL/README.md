Office Space Rental App (OfficeSpaceRentalApp)
This repository contains the JSX implementation of the "officespacerentalapp" project.
It presents real-time data binding demonstrating React element attributes and list looping (`.map()`).
Each listing card displays dynamic properties including name, address, and flat monthly rent.
Custom inline-CSS expressions color-code the pricing dynamically (Red below 60,000, Green at or above 60,000).
Run `npm install` and `npm start` to execute and run this dynamic rentals catalog locally.
