React Blog Portal (BlogApp)
This repository contains the implementation of the "blogapp" project exploring React lifecycle hooks.
It utilizes class components to execute an asynchronous `fetch` request inside the `componentDidMount` lifecycle.
The application actively retrieves real-time JSON mock data and renders titles alongside paragraph content dynamically.
An integrated `componentDidCatch` handler catches execution errors and flags them via alert windows.
Run `npm install` and `npm start` to view the running, populated blog portal on localhost.
