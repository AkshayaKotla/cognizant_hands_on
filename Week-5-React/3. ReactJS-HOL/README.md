Score Calculator Application (ScoreCalculatorApp)
This repository contains the functional component implementation of the "scorecalculatorapp" project.
It features the `CalculateScore` component, which accepts student properties (Name, School, Total, Goal) via props.
The application dynamically calculates the student's average score and formats it inside a styled card.
Custom layout styles are managed cleanly using a dedicated external stylesheet (`mystyle.css`).
Run `npm install` and `npm start` to run the active dashboard locally.
