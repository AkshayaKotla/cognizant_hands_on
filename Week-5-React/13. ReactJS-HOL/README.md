Blogger App (BloggerApp)
This repository contains the layout component suite implemented in the "bloggerapp" project.
It demonstrates four distinct React conditional rendering patterns: if-else, element variables, ternary selectors, and short-circuits.
The modular app structures and renders catalogs for `BookDetails`, `BlogDetails`, and `CourseDetails`.
Every unique layout map loops through raw mock data utilizing distinct, non-index key attributes.
Run `npm install` and `npm start` to run and interact with the combined resources portal locally.
