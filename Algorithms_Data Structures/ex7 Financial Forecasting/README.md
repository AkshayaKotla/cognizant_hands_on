# Exercise 7: Financial Forecasting

## Step 1: Understanding Recursive Algorithms

### Recursion

Recursion is a programming technique in which a method calls itself to solve a problem. It simplifies problems by breaking them into smaller subproblems until a base condition is reached.


## Step 4: Analysis

The recursive algorithm has a time complexity of **O(n)** because one recursive call is made for each year.

The recursive solution can be optimized using techniques such as memoization or by using an iterative approach. These methods reduce unnecessary recursive calls and improve performance for large inputs.
